# Logo-Design Prompts — Kompetenzkompass
**Datei-Typ:** Generative AI Image Prompts (Midjourney / DALL·E / Stable Diffusion / Ideogram)  
**Stand:** März 2026 | **Brand:** Kompetenzkompass — Explainable Matching Platform

---

## Kontext & Designprinzipien

Das Logo soll die Essenz der Marke transportieren:
- **Richtung und Orientierung** (Kompass-Metapher) in komplexen Entscheidungslagen
- **Struktur und Analyse** (Radar-/Wheel-Metapher) für mehrdimensionales Skill-Mapping
- **Klarheit und Präzision** (minimalistische Formensprache) — kein Cartoon, kein Verspieltheit
- **App-Charakter** — das Icon muss als 24×24 px Favicon und als 1024×1024 px App-Store-Icon funktionieren

**Brand-Farben:**
- Ink: `#122620` (Tiefes Dunkelgrün — primäre Symbolfarbe)
- Moss: `#294f3f` (Satiertes Grün — Tiefenwirkung)
- Rust: `#9a3412` (Terrakotta-Akzent — Nadelspitze, Fokusmarker)
- Canvas: `#f5f0e7` (Warm-Creme — Hintergrund)
- Fog: `#e9dfcf` (Helles Beige — zweiter Hintergrund)

no decorative elements, no text. Line weight: medium stroke, consistent.

--ar 1:1 --style raw --v 6

## Prompt 07 — Animated Icon Keyframe (für Lottie / Motion Design)

**Für:** Konzeptskizze an Motion Designer (kein direkter AI-Image-Prompt)

```
Motion design concept for Kompetenzkompass app icon animation (Lottie JSON).

Frame 1 (0ms): Empty radar circle with axis lines, no needle.
Frame 2 (200-400ms): Needle slowly rotates from 9 o'clock position clockwise, 
  easing: ease-out-cubic. The needle sweeps around the radar circle like a scan.
Frame 3 (600-800ms): Needle settles at 12 o'clock (north/up position). 
  Subtle elastic bounce at final position.
Frame 4 (900ms+): Small score marker dot appears on one radar axis — 
  fade-in over 150ms. Color: terracotta (#9a3412).
Frame 5 (1000ms): Idle state. Gentle, very slow 2-degree oscillation of needle.

Principles:
- Total duration: 1.2 seconds for entrance animation
- No loops unless hovering
- Easing: natural, no mechanical movement
- For loading states: continuous slow radar sweep (loop: true, speed: 0.3x)
- Color consistency: matches static icon exactly
- Export: Lottie JSON, 60fps
```

## Verwendungskontext & Technische Exportvorgaben

### Exportierte Dateivarianten (basierend auf docs/landingpage/04-logo-guidelines)

| Dateiname | Format | Maße | Einsatzfeld |
|---|---|---|---|
| `logo-primary-light.svg` | SVG (Vector) | skalierbar | Website Header, Print |
| `logo-primary-dark.svg` | SVG (Vector) | skalierbar | Dunkle Hintergründe |
| `logo-icon-light.svg` | SVG (Vector) | skalierbar | App Icon, Favicon Source |
| `logo-icon-dark.svg` | SVG (Vector) | skalierbar | Dark Mode App Icon |
| `logo-horizontal-light.png` | PNG | 560×140 px @2x | E-Mail-Signaturen |
| `favicon-32.png` | PNG | 32×32 px | Browser Tab |
| `favicon-16.png` | PNG | 16×16 px | Browser Tab (fallback) |
| `apple-touch-icon.png` | PNG | 180×180 px | iOS Homescreen |
| `og-image.png` | PNG | 1200×630 px | Social Sharing |

### Prozessempfehlung

1. **Prompt 01** oder **Prompt 02** in Midjourney/Ideogram eingeben → 4 Variationen generieren
2. Beste 2 Variationen exportieren und in **Figma oder Illustrator** als Vektoren nachbauen
3. Farbcodes aus Brand-Palette exakt übertragen (keine KI-Farbinterpretation übernehmen)
4. Alle 7 Export-Dateivarianten aus Vektorquelle ableiten
5. **Prompt 07** als Briefing an Motion Designer für Lottie-Animation übergeben

### Qualitätskriterien (vor Freigabe prüfen)

- [ ] Symbol erkennbar bei 16×16 px (Favicon-Test im Browser)
- [ ] Kontrast Ink auf Canvas > 10:1 (WCAG AAA)
- [ ] Kontrast Rust auf Canvas > 4.5:1 (WCAG AA)
- [ ] Keine Gradients oder Texturen in der Icon-Version
- [ ] Symbol unverzerrt — perfekte 1:1 Proportionen
- [ ] Dark-Mode-Variante getestet (macOS + Windows Dark Mode)
- [ ] Wortmarke-Lockup: minimale Digitalgröße 140 px Breite
